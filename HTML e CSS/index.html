Exemplos de listas

Lista não ordenada (tag < ul >)

  • Item com a tag < li , sinalizado com um disco (padrão, list-style-type: disc;)
  • Item com a tag < li , sinalizado com um circulo (list-style-type: circle;)
  • Item com a tag < li , sinalizado com um quadrado (list-style-type: square;)

Lista Ordenada (tag < ol >)

  1. Item com a tag < li >, sinalizado com números (padrão, type = "1")
  2. Item com a tag < li >, sinalizado com letras maisculas (type = "A")
  3. Item com a tag < li >, sinalizado com letras minusculas (type = "a")
  4. Item com a tag < li >, sinalizado com com algarismos romanos em caixa alta (type = "I")
  5. Item com a tag < li >, sinalizado com com algarismos romanos em caixa baixa (type = "i")

Imagens (tag < img >)

A tag img tem atributos como src que define de onde a imagem vem(podendo ser arquivo local ou endereço web), e o alt que informa uma legenda para acessibilidade da imagem(caso ela nao carregue e para navegadores com leitura de tela), e tambem o atributo title que define um texto a ser exibido quando o usuario repousa o mouse em cima da imagem

Temos tambem os atributos width="" que adicionado isoladamente, altera a largura mantendo a proporção original, mas se adicionado com o atributo height="" pode distorcer a imagem, isoladamente se comporta como o width, aumentado o tamanho da imagem proporcionalmente
