Imagens (tag < img >)

A tag img tem atributos como src que define de onde a imagem vem(podendo ser arquivo local ou endereço web), e o alt que informa uma legenda para acessibilidade da imagem(caso ela nao carregue e para navegadores com leitura de tela), e tambem o atributo title que define um texto a ser exibido quando o usuario repousa o mouse em cima da imagem

Temos tambem os atributos width="" que adicionado isoladamente, altera a largura mantendo a proporção original, mas se adicionado com o atributo height="" pode distorcer a imagem, isoladamente se comporta como o width, aumentado o tamanho da imagem proporcionalmente

Exemplos de tabelas com a tag < table >

Para criação de tabelas, como elemento pai, devemos usar a tag table, e para cada linha que nossa tabela venha ter, devemos utilizar a tag < tr > e para cada coluna dentro da linha devemos utilizar a tag < td > dentro da row que está dentro da table, tambem é possível manipular altura e largura com os atributos width e height na tag table, junto com a borda do atributo border="".

Exemplos de formulários com a tag < form > e o < input >

Exemplos da tag < iframe >

Esta tag nos permite abrir uma outra página dentro da nossa página, devemos adiconar o atributo src="", que é a pagina destino, podemos tambem definir altura e larguta com o height e with, e tambem se há bordas ou não com o frameboarder="" e o número de px da borda

Exemplos de eventos do mouse como atributo onclick="", onmouseover="", onmouseout=""

Para eventos de mouse, temos tais atributos adicionados a diversas tags, que por sua vez realizan o evento, de acordo com o código javascript disponivel dentro do página e invocado dentro do atributo
