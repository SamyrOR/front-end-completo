● Podemos e tambem devemos, utilizar seletores múltiplos:

"elemento1, elemento2, elemento3 {

font-weight: bold

}"

Aplicando os efeitos de estilo em todos os elementos informados ( atenção para a virgula separando os elementos

● Temos e tambem devemos, utilizar seletores internos:

Código CSS

".elementopai .elementosfilhos {

font-weight: bold

}"

Código HTML

< div class="elementopai">

< span class="elementofilho"> Será aplicado o estilo neste elemento < / span >

< p class="outroelemento"> < span class="elementofilho" > Neste elemento tambem será aplicado < / span > < / p >

< / div >

Aplicando os efeitos de estilo em todos os elementos que estão abaixo do primeiro elemento (elemento pai), independente se está dentro de outro elemento ou não

● Outro seletor, é o por herança, muito similar ao interno:

Código HTML

"elementopai > elementofilho {

font-weight: bold

}"

Código HTML

< div class="elementopai">

< span class="elementofilho"> Será aplicado o estilo neste elemento < / span >

< p class="outroelemento"> < span class="elementofilho" > Neste elemento o estilo não será aplicado < / span > < / p >

< span class="elementofilho"> Mas neste tambem < / span >

< / div >

Porem, este aplica o efeito de estilo somente e diretamente no elemento filho especifico, sem atingir os elementos que são filhos de outros

● Complementando os tipos de seletores, há o seletor adjacente:

Código CSS

"elementoprecessor + elementoaseraplicado {

font-weight: bold

}"

Código HTML

< div class="elementoprecessor">

< span class="elementoaseraplicado"> Será aplicado o estilo neste elemento < / span >

< p class="outroelemento"> < span class="elementofilho" > Neste elemento o estilo não será aplicado < / span > < / p >

< span class="elementofilho"> Neste elemento tambem não será aplicado < / span >

< / div >

Este seletor nos permite aplica estilos ao elementos após o sinal +, no código HTML o estilo será aplicado quando o elemento for o primeiro seguido do predecessor, se não for encontrado, o estilo não é aplicado

● Por último tambem temos o seletor hover:

Código CSS

"elemento:hover{

font-weight: bold

}"

Ao passar o mouse por cima do elemento que tem o seletor hover, os estilo do elemento será mudado de acordo com o desejado durante o hover no css

Um recurso muito utilizado dentro do CSS é o atributo "position", como propriedades temos:

O position: static; que é o padrão quando nenhum outro é declarado, respeita a ordem de estrutura da página

O position: absolute; este dependendo da situação é bastante usado, ele fica flutuando, acima da tela, sobrepondo qualquer elemento. Flutua em posição relativa do elemento anterior a ele

O position: fixed; esta propriedade deixa fixo na tela o elemento tambem sobrepondo outros elementos, e acompanha a barra de rolagem po toda tela.

O position: relative; é semelhante ao static porem conseguimos mudar a profundidade e até alterar os elementos.

Outro recurso tambem de suma importância é o atributo "display":

O display: none; faz com que o elemento não seja mostrado na tela, como se nem existisse.

O display: inline; esta propriedade faz com que o elemento se adapte ao seu conteúdo (impedindo alterações no tamanho e largura), e se mostre na mesma linha de outros elementos inline (como a tag < span >).

O display: block; que faz com que o elemento seja mostrado como um bloco, separando-se de outros elementos, podendo ser trabalhado realmente como um bloco, se alterando o seu tamanho, largura, posição, etc...

O display: inline-block; por fim, está propriedade nos permite unir a renderização em uma mesma linha como o inline e a customização do atributo como um bloco.

Podemos através do CSS formatar texto, seguem alguns atributos para a edição dos mesmos:

.texto {

color: cor; (muda a cor do texto)

font-size: 10px; (muda o tamanho do texto)

font-family: Font; (muda a familia da fonte do texto, podendo conter mais de uma)

font-weight: 300; (muda a expessura da font do texto (negrito com bold))

text-align: center; (alinha o texto dentro do espaço de acordo com a posição desejada)

text-transform: uppercase; (possibilita alteração no texto, deixando-o o em caixa alta, caixa baixa, etc)

text-decoration: underline; (adiciona decorações no texto, como o underline(sublinhado))

}
