O HTML é equivale a estrutura da página e o CSS o acabamento, temos 3 tipos diferentes de declarar o estilo da página, sendo o CSS inline (com o atributo style="" definir o estilo diretamente na tag do elemento, a tag style dentro da tag head , e o arquivo css externo chamado via tag link, toda cor declarada no css pode ser via rgb (), hexadecimal #ff00ff e escrita em inglês "red"

Exemplo de estilo inline com a tag style="color: red;" alterando a cor do texto para vermelho

Na estrutura dos arquivos CSS externos, temos os seletores, que são divididos em nome do seletor, classe do elemento(.nomedaclasse) e id (#nomedoid), sendo que, se utilizado o nome do elemento, todos os elementos correspondentes serão alterados de acordo com o estilo, se utilizarmos a classe, apenas os elementos contidos na classe terão tal estilo, e devemos evitar o uso do ID, pois é um identificado único do elemento que trás as informações do elemento

Identificador pelo elemento < span > ; Identificador pela classe .class="exemplo2" ; Identificador pelo ID, #exemplo3

Dentro do CSS podemos lidar com tags que tem formas de se mostrar, como a tag < div > que funciona como um bloco, se se parando de qualquer conteúdo a sua volta, e do outro lado temos tags como a < span > que se mostra na mesma linha que outros componentes

Tambem teremos as bordas do elemento, praticamente todo elemento pode ter borda, que pode ser definida como border: tamanho estilo cor;

Exemplo de borda 1px solid red Exemplo de borda 2px dotted blue Exemplo de borda 3px dashed yellow Exemplo de borda 4px double green

Podemos e tambem devemos, utilizar seletores múltiplos:

"elemento1, elemento2, elemento3 {

font-weight: bold

}"

Aplicando os efeitos de estilo em todos os elementos informados ( atenção para a virgula separando os elementos

Temos e tambem devemos, utilizar seletores internos:

"elementopai elementofilho {

font-weight: bold

}"

Aplicando os efeitos de estilo em todos os elementos que estão abaixo do primeiro elemento (elemento pai), independente se está dentro de outro elemento ou não

Outro seletor, é o por herança, muito similar ao interno:

"elementopai > elementofilho {

font-weight: bold

}"

Porem, este aplica o efeito de estilo somente e diretamente no elemento filho especifico, sem atingir os elementos que são filhos de outros
