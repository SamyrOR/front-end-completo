Arquitetura de CSS BEM

BEM é uma arquitetura de CSS que traz diversos benefícios, a convenção de nomeação das classes, modularização, reusabilidade, facil leitura e entendimento e um css bem estruturado

BEM = Block Element Modifier

Para a arquitetura temos algumas convenções

Block

Significa o bloco, onde os elementos são dispostos, como um menu que tem diversos items

.bloco

Em blocos nao podemos usar dependência hierárquica, e tambem devemos apenas usar CLASS (.), sem id ou tag

Element

O elemento que se encontra dentro deu um bloco

.bloco__elemento

Todo elemento é tem sua class com o primeiro nome ao bloco que se refere, seguido de 2 underlines identificando um elemento

Se temos elemento dentro de elementos em um bloco, utilizamos sempre apenas o nome do bloco e do elemento em questão .bloco_elemento-filho e não .bloco__elemento-pai__elemento--filho

Modifier

Significa o modificador, como uma cor, um estado, um tamanho, etc, tudo que modifica o elemento ou o bloco para um outro estado alem do natural

.bloco--modificador

ou

.bloco__elemento--modificador

O modificador é sempre precedido por dos hifens -- identificando como um modificador

Considerações

Todo identificador deve ser independente e não extremamente especifico
