NOVAS TAGS HTML

No HTML5, o ultimo padrão do HTML, temos novas tags que melhoram tanto o SEO, quanto a acessibilidade da nossa página, são algumas delas:

A < header > ( não confundir com a tag head). Está tag nos possibilita deixar semântico o cabeçalho da nossa página

A tag < nav >, deve ser utilizada para identificar o nosso principal menu de navegação do site (geralmente utilizada aninhada a tag < header >

A tag < footer > nos permite definir de forma semântica o rodapé da nossa página

A tag < main > que define todo o conteúdo principal da página

A tag < article > que deve ser melhor utilizada em artigos, partes da página que são únicas e independentes e sem reslação com outras partes

A tag < section > por outro lado nos permite agrupa vários itens correlacionados e definir diversas secções da nossa página

A tag < time > com o atributo datetime="ano/mes/dia hora", ajuda a traduzir tempos no nosso texto, para que a máquina saiba a data comentada.

O HTML5 tambem nos permite a criação de TAGs personalizadas (nos mesmo colocamos o nome, basta abrir e fechar como qualquer outra tag )e que funcionam, tanto no navegador quanto a aplicação de destilo

HTML5 tambem nos trouxe diversos novos e muito úteis tipos de input, sendo eles:

Campo de pesquisa com "X" para apagar o campo.

Campo que ja valida se o texto digitado é um e-mail.

Campo que auxilia acessibilidade mostrando apenas o teclado numérico e identifica que o valida digitado é um número de telefone.

Campo que ja identifica que o valor é e um número, impede a digitação de letras, adicionando já tambem a funcionalidade de subir ou descer o número com setas no fundo do campo

Campo do tipo range, que nos permite adicionar um número mínimo e um máximo (min="" max="") e setar dentro destes valores com um campo de range

Campo muito útil que ja nos permite identificar e facilmente selecionar datas

Campo da família date que permite selecionar meses

Campo da família date que permite selecionar semanas

Campo da família date que permite selecionar a data e a hora de uma vez.

Campo que nos permite selecionar cores

Por fim o atributo "required" nos permite impedir o envio de um form que não o campo requerido não esteja preenchido de acordo

Foi tambem adicionado o atributo placeholder="texto" para os inputs, que nos permite adicionar algum texto dentro do input sem alterar o seu funcionamento e facilitando a identificação do tipo de input. O atributo "autofocus" que nos permite que assim que a página esteja carregada, o input com tal atributo seja focado no navegador do usuário. O atributo size que permite definir o tamanho do input na tela do usuário(pouco utilizado). Outro bom atributo é o maxlength="numero", que nos permite definir uma quantidade máxima de caracteres a serem inseridos no input. Há tambem o autocomplete="on/off", que nos permite ativar ou desativar a funcionalidade de auto completar o texto no input

Atributo data

Temos a inovação do atributo data-nomedesejado, que no s permite adicionar informações dentro do elemento HTML que podem ser resgatadas e utilizadas pelo JavaScript e que ficam apenas no código sem transparecer para o usuário. Recurso muito utilizado por diversos programadores, podendo ser inserido em qualquer tag do HTML.

Tag < video >

Esta tag como o o próprio nome nos sugere, é utilizada para inserir vídeos na nossa pagina sem ajuda de recurso de terceiros como o falecido flash

A tag deve ser utilizada junto de atributos, sendo estes, os de tamanho (width="valor", height="valor"), de controle (controls) que mostra para o usuário um painel com controles do video, "autoplay" como tambem o nome sugere, assim que carregada a página o video ja inicia, o "loop" que assim que o video é finalizado volta a tocar novamente. Dentro da tag < video > devemos inserir a tag < source >, que será a fonte do nosso video com o atributo src="caminho" e o type="video/formato", atualmente sendo suportado 3 tipos de formatos, o mp4, o ogg e webm. Podendo ser adicionada mais de uma tag source caso o browser do usuário não suporte um dos formatos e um texto que irá aparecer caso apenas nenhum dos formatos seja suportado

Tag < audio >

Semelhante a declaração da tag < video > a tag audio nos permite a adicionar arquivos de aúdio na nossa página, com as mesmas opções de atributos que a tag < video >, adicionando apenas a muted que inicia o audio com o som mutado e os tipos de arquivos suportado, sendo eles "audio/mpeg" (o mp3), o "audio/ogg" e o "audio/wav"

O Media query (@media)

Esta propriedade do CSS3, é a principal responsável pela possibilidade de criação de design responsivo da nossa página, é a propriedade que altera os estilos css de acordo com a orientação passada, e pra qualquer estilo que desejar.

Design Responsivo

Design responsivo, é a capacidade da nossa página em se adequar em diferentes tipos de aparelhos (telas), sendo adaptável para qualquer tamanho de tela, passando por celulares, tablets, notebooks, monitores e até televisores.

Com algumas configurações básicas, como o Media Query (@media), e definição de tamanhos com % (width: 100%;), este responde de forma adaptada ao tamanho da tela da qual se encontra, essencial para conhecimento do programador front-end, pois inúmeros usuários tambem se encontram em outras dispositivos além do pc como ja se foi um dia.

Complementando o design responsivo, para que as funcionalidades sejam realizadas em todos os tipos de aparelhos, devemos usar uma tag < meta > dentro da tag do header, com as seguintes propriedades:

name="viewport" content="width=device-width, initial-scale=1.0"

Que fazem com que o conteúdo seja ajustado do tamanho da tela do aparelho permitindo a resposividade de acordo com programação.

normalize.css

Um item altamente recomendado para o desenvolvimento de qualquer página, é o uso de css resets, como o normalize.css, resumidamente, ele padroniza todo o css que vem pre definido de cada navegador, diminuindo assim a margem para erros de layout devido a diferença de navegadores.

Degradê

Para criar efeitos de degradê com o CSS3, podemos utilizar o atributo linear-gradient(posição em graus, primeira cor, segunda cor, etc) exemplo:

background: linear-gradient(180deg, red, blue)

Animações

Graças ao CSS3 podemos animar elementos apenas com CSS, utilizando @keyframes, segue exemplo do código:

@keyframes giratorio{

0% {propriedade(s) a ser alterada}

25% {propriedade(s) a ser alterada}

50% {propriedade(s) a ser alterada}

75% {propriedade(s) a ser alterada}

100% {propriedade(s) a ser alterada}

}

E no elemento que devemos utilizar a animação, podemos atribuir a propriedade animation: nomedaanimação duração velocidadedaanimação delay repetições direção(pra frente, de trás pra frente ,etc)

animation: giratorio 2seg linear 3seg infinite normal

Caso uma das propriedades não sejam setadas, será tomado o valor padrão como base

Sombras

Podemos aplicar sombras através do CSS3 , em elementos e textos com a seguinte sintaxe:

Prefixos vendor

Cada navegador possui um motor de renderização, fazendo que cada um interprete a sua página a sua maneira, o que pode causar diferenças na página de navegador para navegador:

IE = Trindet

Edge = EdgeHTML

Firefox = Gecko

Opera = Webkit

Safari = Webkit

Chrome = Webkit

Para corrigir comportamentos estranhos, podemos e devemos utilizar os prefixos vendor, de cada motor gráfico, para garanti o comportamento desejado

-ms- (Internet Explorer)

-moz- (Firefox)

-o- (Versões antigas do Opera)

-webkit- (Chrome, Safari, versões mais recentes do Opera.)

Word Wrap

Nesta propriedade do CSS3 nos possibilita de maneira eficaz, enquadra o texto dentro da box que o mesmo se encontra, evitando que se ele for muito longo, estoure a caixa de texto, basta adicionar a seguinte linha:

word-wrap: break-word;
