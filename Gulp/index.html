Gulp

Gulp é um task runner que serve para facilitar tarefas do dia a dia como minificar arquivos, converter arquivos,e reduzir tamanho de imagens

Sua sintaxe é bem simples, basta instalar o gulp-cli globalmente e na pasta de seu projeto instalar o gulp localmente com --save-dev

Após instalado basta adicionar uma arquivo chamado gulpfile.js, dentro iremos adicionar as task, com functions, e para poder rodar a task, basta exporta-la com export.nome-da-task = task

Para rodar, informamos no terminal apenas gulp nome-da-task, caso o nome nao seja informado, o mesmo sempre procura-ra pela

Como podemos ter diversas tasks, temos como roda-las em paralelo ou em serie

Para utiliza-los, devemos antes de tudo importa-los com, const {series, parallel} = require('gulp')

Series

exports.nome-da-task = series(tasks, tasks, task), faz com a chamada de uma task, rode tasks definidas em sequencia

Parallel

exports.nome-da-task = parallel(tasks, tasks, task), faz com a chamada de uma task, rode tasks definidas em paralelo, todas ao mesmo tempo

Podemos tambem utilizar ambos aninhados, como dentro de uma series, rodar uma parallel

src pipe e dest

A função src(caminho/dos/arquivos), é a função responsável por localizar os arquivos

As funções pipe(...), são as funções responsáveis realizar as atividades e utilizar os plugins dentro de uma task, podendo ser mais de um pipe

A função dest, é a função responsável salvar os arquivos no destino, é utilizada dentro de um pipe = .pipe(dest(destino/dos/arquivos))
