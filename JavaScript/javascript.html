Variáveis

É a declaração de armazenamento de algum dado na memoria do computador, que pode ser reutilizado sendo chamado com o nome declarado, exemplo de declaração variável:

var nomaDaVariavel = "Algum valor"

Os valores podem ser strings que são declaradas entre aspas, números que não precisam de aspas, undefined valor não definido(razão de muitos bugs), null elemento vazio

Temos tambem a declaração de let, que é um tipo de variavel disponivel apenas em no escopo em que é declarada, e const, que recebe um valor e não pode/deve ser alterado

Podemos realizar operações matemáticas com as variaveis, com os operadores:

+ para soma

- para subtração

* para multiplicação

/ para divisão

% chamado de modulo para identificar o resto da divisão

Arrays

Array é uma variavel que carrega um conjunto de informações, ex:

var lista = ["Arroz" , "Feijão", "Salada", "Bife" , "Batata frita"]

Declarado com colchetes, cada item separado por virgula equivale a um dado, para obte-los podemos chamar a variavel informando em qual casa o item que desejamos se encontra, começando a contagem do 0

lista[0], retornará o "Arroz"

podemos manipular Arrays com algumas propriedades como

lista.push('item'), adiciona o item no final do array

lista.pop(), remove e retorna o ultimo item do array

lista.shift(), remove e retorna o primeiro item do array

lista.indexOf('item'), retorna a posição em que se encontra o item informado no parametro, caso o item não seja encontrado retorna -1

lista.join('/'), concatena todos os items do array com o caracter informado como parametro

Condicionais

Uma condicional, reage com condição imposta, o verdadeiro "se" e "se não", sintaxe:

if (condição) {

código a ser executado caso a condição seja verdadeira

} else {

Código a ser executado caso a condição seja falsa

}

Laços de repetição

Na linguagem de programação javascript, temos dois tipos de laços de repetição, o for e o while, os laços de repetição executam determinado bloco de código enquanto a condição for verdadeira:

O for recebe como parametro a variavel, a condição e a execução:

for(var x = 0; x < 10; x++) {

código a ser executado até que x seja > que 10.

}

O while executa a mesma operação, porem é declarado de forma diferente, ja temos que ter a variavel declarada antes

x = 0

Então executamos o código

while(x < 10) {

código a ser executado enquanto a condição seja verdadeira

}

Manipular o HTML

Com javascript é possível manipular dados do HTML da página, como adicionar e remover elementos, alterar o conteúdo de elementos etc

Basta atribuir o elemento como valor a uma variavel, usando document.querySelector('seletor css (podendo ser tag 'div', id #, e class ., e qualquer outro seletor css porem sempre entre aspas) ')

ex: let button = document.querySelector('.button')

Após selecionado podemos utilizar propriedades como innerHTML, etc.

Programação orientada a eventos

Em javascript programamos muito orientado a eventos, ou seja, se algo ocorrer estamos ouvindo o envento acontecer, e quanto contecer executamos determinada função

Eventos de mouse:

click = quando o elemento é clicado

dblclick = acionado quando se clica duas vezes rapidamente

contextmenu = menu disposto no botão direito

mousedown = quando botão esquerdo é clicado

mouseup = quando se solta o click do botão esquerdo

mouseover = quando o mouse passar pelo elemento

mouseout = quando o mouse passa por cima e sai do elemento

mousemove = captura o movimento do mouse em cima do elemento

Eventos de teclado

Para eventos de teclado temos apenas 3 opções

keydown = quando o botão é apertado

keyup = quando eu solto o botão

keypress = enquanto o botão é pressionado (segurado)

Mas devemos capturar o botão que foi apertado para executar determinadas ações, fazemos isso passando o evento como paramentro, e usando propriedades com event.key para identificar a tecla

Podemos tambem saber se o shift, ctrl ou alt estavam pressionadas com as propriedades event.shiftKey, .ctrlKey, .altKey, todas retornam um valor booslean

Evento de página

Evento roda quando algo ocorre na página

onload = quando a página termina de ser carregada

unload = quando a página é fechada

Eventos de formulário

submit = quando o formulário é enviado

focus = quando um elemento é selecionado

blur = quando um perde a seleção

change = quando o elemento tem o sue valor trocado
